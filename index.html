Troubleshooting Guide

build Common Solutions

dns Database Connection Failed

Error: "Connection failed: Access denied for user..."

Solution: Open includes/db.php and verify your MySQL username and password. Ensure XAMPP MySQL service is running.

mark_email_read Emails Not Sending

Error: "SMTP Error: Could not connect to SMTP host."

Solution: Check includes/GmailMailer.php. Ensure you are using a valid Google App Password, not your regular Gmail password. Verify port 465 is open.

image_not_supported Images Not Loading

Issue: Product images appear broken.

Solution: Ensure the assets/images/ directory exists and has read permissions. Check if the image filename in the database matches the actual file (case-sensitive on Linux).

lock_clock Session/Login Issues

Issue: User is logged out immediately after logging in.

Solution: Check php.ini for session.save_path. Ensure the directory is writable. Clear browser cookies and try again.
